# Nyra Dashboard User Guide

Welcome to the Nyra Dashboard! This guide explains each feature and how to use it.

---

## Growth Tracker Panel
- Shows Nyra’s relationship level, technical knowledge, sysadmin readiness, and personality.
- Updates automatically as Nyra evolves.

## Avatar Panel
- Displays Nyra’s animated avatar.
- Avatar switches between idle and speaking states.
- Status indicator shows if Nyra is ready or talking.

## Chat Panel
- Main area for chatting with Nyra.
- Type messages and click “Send” or press Enter.
- Nyra’s responses appear in the chat log.
- Scroll to review conversation history.

## Action Buttons
- **Toggle Mode:** Switch between text and voice input.
- **Clear:** Wipe the chat log.
- **Save:** Download the chat log as a text file.
- **Settings:** Open the settings modal.

## Settings Modal
- **Dark Mode:** Toggle between light and dark themes.
- **Enable Notifications:** (Coming soon) Get notified of new messages/events.
- Click “Close” or the backdrop to exit settings.

## Voice Input (if enabled)
- Switch to voice mode and click the microphone button.
- Speak your message; Nyra will transcribe and respond.
