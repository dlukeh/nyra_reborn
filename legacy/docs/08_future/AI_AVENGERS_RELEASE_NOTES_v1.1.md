## 🧠 Core Enhancements

| Module | Purpose | Description |
|--------|----------|-------------|
| `memory_consolidation.py` | 💤 Night Ops | Generates structured nightly reflections in JSON format. |
| `generate_morning_report.py` | 🌅 Morning Sync | Converts reflections into human-readable Markdown morning reports. |
| `/api/run_night_ops` | API Trigger | Manually or automatically runs Night Ops. |
| `/api/generate_morning_report` | API Trigger | Allows Nyra or the user to generate the morning report via Flask. |
| `ai_avengers_morning_report_template.md` | Template | Defines report structure and placeholders. |
